# vim-colors

A set of vim color schemes collected from various places.

## Install

###### Manually
```sh
cp -R colors/ $HOME/.vim/colors
```

###### Plug

```vim
Plug 'ivanrad/vim-colors'
```

Reload `.vimrc` and `:PlugInstall` to install.

See [latest-vimcolors](latest-vimcolors) for details.
